[FR] La commande /chatLog permet d'enregistrer toutes les conversations du jeu y compris les messages système et les emotes dans un fichier texte (sauvegardé dans le répertoire "...\Ryzom\save").
Ce menu permet de charger ce fichier pour voir les conversations en couleur, de filtrer les messages et de les copier au format HTML, BBCode ou texte brut.
[EN] The command /chatLog allows to record all the conversations of the game including system messages and emotes in a text file (saved in the directory "...\Ryzom\save").
This menu allows to load this file to see the conversations in color, to filter messages and to copy them in the HTML format, BBCode or plain text.
[DE] Der Befehl /chatLog erlaubt, alle Gespräche des Spiels zu registrieren, das System-Nachrichten einschließt, und reagiert in einer Text-Datei (gerettet im Verzeichnis "...\Ryzom\save") emotional.
Dieses Menü erlaubt, diese Datei zu laden, um die Gespräche in Farbe zu Filternachrichten zu sehen und sie im HTML-Format, BBCode oder einfachem Text abzuschreiben.
